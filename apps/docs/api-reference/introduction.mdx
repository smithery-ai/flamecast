Flamecast exposes a REST API for managing agents and templates, and a WebSocket endpoint for real-time session interaction.

## Base URL

```
http://localhost:3001/api
```

## Authentication

Flamecast does not currently implement authentication. All endpoints are open.

<Warning>
  Do not expose a Flamecast instance to the public internet without adding your own auth layer.
</Warning>

## API surface

| Area | Description |
|---|---|
| [REST endpoints](/api-reference/endpoints) | Create, list, and terminate agents and templates |
| [WebSocket](/api-reference/websocket) | Real-time session events and control |
| [Client SDK](/api-reference/client-sdk) | TypeScript client with auto-reconnecting WebSocket |

## Typed client

Flamecast ships a typed HTTP and WebSocket client at `@flamecast/sdk/client`:

```typescript
import { createFlamecastClient } from "@flamecast/sdk/client";

const client = createFlamecastClient({ baseUrl: "http://localhost:3001" });

const templates = await client.fetchAgentTemplates();
const agent = await client.createSession({ agentTemplateId: "example" });
```

See the [Client SDK reference](/api-reference/client-sdk) for the full API.
