<Info>
  **Prerequisites**: Node.js 20+ and pnpm installed.
</Info>

<Steps>
<Step title="Clone and install">

```bash
git clone https://github.com/anthropics/flamecast.git my-app
cd my-app
pnpm install
```

<Note>
  ***Coming soon:*** `npm create flamecast@latest` will scaffold a new project automatically.
</Note>

</Step>

<Step title="Start the dev server">

```bash
pnpm dev
```

This starts the API server on port 3001 and the React UI dev server.

</Step>

<Step title="Launch a session">

Open **http://localhost:3000**. The home page lists registered agent templates. Click **Start session** to launch one.

You can also create a session from the command line:

```bash
curl -X POST http://localhost:3001/api/agents \
  -H "Content-Type: application/json" \
  -d '{ "agentTemplateId": "example" }'
```

</Step>
</Steps>

## Use the SDK directly

You can also create a Flamecast instance programmatically:

```typescript
import { Flamecast } from "@flamecast/sdk";
import { createPsqlStorage } from "@flamecast/storage-psql";

const flamecast = new Flamecast({
  storage: await createPsqlStorage(),
});

await flamecast.listen(3001);
```
